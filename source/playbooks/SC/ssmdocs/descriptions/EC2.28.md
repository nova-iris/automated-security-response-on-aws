### Document Name - ASR-SC_2.3.0_EC2.28

## What does this document do?
This document enables EBS encryption by default for the AWS account in the current region. All new EBS volumes will be encrypted.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
