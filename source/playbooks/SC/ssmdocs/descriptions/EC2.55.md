### Document Name - ASR-SC_2.3.0_EC2.55

## What does this document do?
This document configures a lifecycle policy on an Amazon ECR repository to manage image retention and cleanup of old images.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
