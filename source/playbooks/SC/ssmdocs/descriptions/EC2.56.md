### Document Name - ASR-SC_2.3.0_EC2.56

## What does this document do?
This document enables image scanning on push for an Amazon ECR repository to automatically scan container images for vulnerabilities.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
