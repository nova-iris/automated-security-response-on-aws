### Document Name - ASR-SC_2.3.0_EC2.57

## What does this document do?
This document creates VPC interface endpoints for AWS Systems Manager (SSM) services (ssm, ssmmessages, ec2messages) in a VPC to enable private connectivity.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
