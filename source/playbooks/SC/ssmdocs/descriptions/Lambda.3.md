### Document Name - ASR-SC_2.3.0_Lambda.3

## What does this document do?
This document provides guidance for configuring VPC settings on Lambda functions. VPC configuration requires careful planning for network access, so manual remediation may be required.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
