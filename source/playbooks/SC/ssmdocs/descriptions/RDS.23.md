### Document Name - ASR-SC_2.3.0_RDS.23

## What does this document do?
This document checks and provides guidance for changing RDS instance default ports. Port changes require planning for application connectivity, so manual remediation may be required.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
