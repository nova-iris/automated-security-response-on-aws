### Document Name - ASR-SC_2.3.0_RDS.9

## What does this document do?
This document enables publishing of database logs (error, general, slow query, audit) to Amazon CloudWatch Logs for an RDS DB instance.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
