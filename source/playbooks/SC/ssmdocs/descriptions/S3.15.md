### Document Name - ASR-SC_2.3.0_S3.15

## What does this document do?
This document checks S3 Object Lock configuration. Object Lock must be enabled at bucket creation time and cannot be added to existing buckets, so this provides manual remediation guidance.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
