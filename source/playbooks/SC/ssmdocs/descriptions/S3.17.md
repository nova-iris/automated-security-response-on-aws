### Document Name - ASR-SC_2.3.0_S3.17

## What does this document do?
This document configures S3 bucket default encryption to use AWS KMS (SSE-KMS) instead of SSE-S3, using a customer-managed KMS key.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
