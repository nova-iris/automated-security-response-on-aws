### Document Name - ASR-SC_2.3.0_S3.20

## What does this document do?
This document checks S3 MFA Delete configuration. MFA Delete requires root account credentials and cannot be fully automated, so this provides manual remediation guidance.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
