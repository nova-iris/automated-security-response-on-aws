### Document Name - ASR-SC_2.3.0_S3.7

## What does this document do?
This document enables cross-region replication on an S3 bucket. It verifies versioning is enabled and configures replication rules.

## Input Parameters
* Finding: (Required) Security Hub finding details JSON
* AutomationAssumeRole: (Required) The ARN of the role that allows Automation to perform the actions on your behalf.

## Output Parameters
* Remediation.Output
